# 🩺 MediBridge AI

AI-powered medical report assistant that explains complex reports in simple language.

---

## 🚀 Features

- 📄 Upload medical reports (PDF)
- 🧠 AI-based analysis using Mistral (Ollama)
- 🔍 Works on real-world reports (OCR supported)
- ⚠️ Detects abnormal values
- ❓ Generates doctor questions
- 🧾 Suggests next steps

---

## 🧠 Tech Stack

- Python
- Streamlit
- Mistral (Ollama - Local LLM)
- FAISS (Vector Search)
- EasyOCR
